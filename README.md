# Hull-Tactical-Market-Prediction
The Hull-Tractical-Market-Prediction is a kaggle competition project.

## Setup
### Prerequisites
* Python 3.10+
### Installation
1. Install the dependencies:
```
pip install -r requirements.txt
```
2. Install the virtual environment and package
```
conda create --prefix ./marker python=3.10 -y

pip install -e .
```
